# Terrain Beta

Welcome to Terrain Beta! This is a **Bedrock Edition** Add-on inspired by [Moderner Beta](https://modrinth.com/mod/moderner-beta), dedicated to recreating world terrain generation from multiple classic Minecraft versions.

## Features
- Recreates world generation from several older versions, such as **Alpha 1.0.16_02**, **Beta 1.7.3**, as well as some special generators (such as **Beta 1.7.3 Sky Dimension**).
- Provides custom options to configure the **Seed**, **World Height Offset**, and **Far Lands Start Coordinate**, and lets you toggle features like **Ores**, **Trees**, **Flora**, **Springs**, **Snow**, **Caves**, **Lakes**, **Winter Mode**, and **Random Features** depending on the selected generator.

## Additional Suggestions

For a more authentic Beta-style visual experience:

- Use the resource pack [Classic Texture Fix](https://www.curseforge.com/minecraft-bedrock/texture-packs/classic-textures-fix) to restore classic textures.
- Set your **render distance to 5 chunks** to better match the distinct feel of early versions.

## Notes

- No experimental gameplay features are required, and it is compatible with achievements.

> **Warning**
>
> It is strongly recommended to run this Add-on on a high-performance device. Even then, you may still encounter slow chunk loading. If you see chunks that have not fully loaded yet, wait a few seconds until they finish loading.
>
> **Known issue**: Some cross-chunk features (such as trees) may generate incompletely due to a bug.
